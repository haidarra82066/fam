# fam

Next.js App Router + TypeScript foundation for the family tree app.

## Setup

1. Copy env file:

```bash
cp .env.example .env.local
```

2. Install dependencies:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```

## Routes scaffolded

- `/`
- `/login`
- `/signup`
- `/pending-approval`
- `/dashboard`
- `/tree/[treeId]`
- `/admin/access-requests`
